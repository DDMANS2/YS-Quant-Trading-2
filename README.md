# 📈 YS Quant Trading

코스피·코스닥 종목 중 일봉 종가가 N일 이동평균선을 돌파한 종목을 자동 탐지하고, PER·PBR·ROE·PEG 가치 지표와 증권사 컨센서스 목표가까지 한 화면에서 확인하는 무료 한국 주식 스크리너입니다.

🌐 **라이브 데모**: https://ys-quant.streamlit.app

---

## 주요 기능

### 🔍 이평선 돌파 스크리닝
- 코스피·코스닥 종목 중 일봉 종가가 N일(20/60/100/200/300일) 이동평균선을 돌파한 종목 자동 탐지
- 시가총액 상위 N개로 스캔 범위 좁히기 (KOSPI/KOSDAQ 각각 100~전체)
- 돌파 시점 / 최소 거래량 / 최소 주가 필터
- 병렬 처리 + 자동 재시도로 빠르고 안정적인 스캔

### 💎 가치 지표
검색 결과 종목의 펀더멘털을 한 화면에서:
- **PER · PBR** (가장 최근 실적연도 기준)
- **ROE** (시계열에서 최신 실적연도 추출)
- **PEG** (PER ÷ EPS 성장률, 자체 계산)
- **EPS · BPS · 배당수익률 · 추정 PER**

### 🎯 증권사 컨센서스
- 평균 목표주가 + 현재가 대비 상승여력
- 투자의견 (강력매수/매수/중립/매도)
- 컨센서스 기준일 + 최근 증권사 리포트 5건 (증권사명·제목·날짜)

### 📊 상세 차트
- **캔들 차트** + 이동평균선 (5/20/60/120) + 볼린저밴드(20, 2σ)
- 거래량 / MACD(12,26,9) / RSI(14) 보조지표 토글
- 마우스 휠 확대/축소, 드래그 패닝, 더블클릭 리셋
- 모드바에서 PNG로 저장 가능

### 📐 인포그래픽 디자인
- KOSPI/KOSDAQ/KOSPI200 지수 카드 (30거래일 스파크라인 포함, 클릭 시 네이버 증권 연동)
- 검색 결과 시장 분포 도넛 + 평균 등락률/ROE/목표가 상승여력 진행 막대
- 종목명·종목코드 클릭 시 네이버 증권 종목 페이지 새 탭 이동

### 📱 반응형
- 데스크톱(≥1025px) / 태블릿(≤1024px) / 모바일(≤768px) / 모바일 세로(≤480px) 4단계 적응
- 모바일에서도 차트 패닝·줌, 카드 그리드 자동 단일 컬럼

## 데이터 출처 (모두 무료, API 키 불필요)

| 데이터 | 소스 |
|---|---|
| 종목 리스트 + 시가총액 | FinanceDataReader (1차) → 네이버 모바일 stock JSON (fallback) |
| 일봉 OHLCV | pykrx (KRX 공식) |
| 시장 지수 | FinanceDataReader |
| 가치 지표 + 컨센서스 + 리포트 | 네이버 모바일 stock JSON |

## 로컬 실행

```bash
git clone https://github.com/DDMANS2/YS-Quant-Trading-2.git
cd YS-Quant-Trading-2
pip install -r requirements.txt
streamlit run app.py
```

`http://localhost:8501`이 자동으로 열립니다.

---

## 사용 가이드

1. 사이드바에서 **시장**(KOSPI/KOSDAQ), **이동평균선 기간**(기본 300일), **시총 상위 N**(기본 200+200) 선택
2. **🔍 스크리닝 시작** 클릭 → 약 30초~3분 소요
3. 결과 테이블에서 종목명/종목코드 클릭 시 네이버 증권으로 이동
4. **종목 상세 보기**의 selectbox로 종목 선택 → 차트 + 가치 지표 + 컨센서스 확인
